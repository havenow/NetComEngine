# NetComEngine

- 用socket API建立简易TCP服务端		    
1、建立一个socket		     
2、绑定接受客户端连接的端口 bind		     
3、监听网络端口 listen		      
4、等待接受客户端连接 accept		      
5、向客户端发送一条数据 send      
6、关闭socket closesocket		     

- 用socket API建立简易TCP客户端端		
1、建立一个socket		     
2、连接服务器 connect		     
3、接收服务器信息 recv		     
4、关闭socket closesocket
